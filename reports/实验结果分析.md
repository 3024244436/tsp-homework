# TSP-D 实验结果分析

> 对照论文：Bouman, Agatz, Schmidt. *Dynamic programming approaches for the traveling salesman problem with drone*. Networks, 2018.

---

## 一、实验数据说明

| 来源 | 实例数 | n 范围 | 分布类型 |
|------|--------|--------|----------|
| gen_instances.py 生成 | 9个 | 8, 10, 12 | uniform / 1-center / 2-center 各3个 |
| TSPLib 标准库 | 2个 | 14 (burma14), 16 (ulysses16) | 真实地理坐标 |
| FSU 标准测试集 | 1个 | 15 | 标准 TSP 测试集 |

**算法配置：**
- DP：精确动态规划（k = ∞），三遍 Pass 1/2/3，复杂度 O(3ⁿ·n²)
- RDP：限制型动态规划，L ∈ {1, 2, 3, 4}，限制每次 operation 中 truck-only 节点数 ≤ L
- 固定参数：truck_speed = 1.0，drone_speed = 2.0（α = 2），drone_endurance = 10⁹（无限续航）

## 六、论文结论逐条验证

| # | 论文结论（§4） | 验证状态 | 本实验证据 |
|---|-------------|:------:|-----------|
| 1 | "DP 可求解比 IP 更大规模的实例" | ✅ | n=16 ulysses16 在 11.3s 内精确求解（论文 IP 仅到 n=10） |
| 2 | "限制 k 显著降低求解时间" | ✅ | n=16: 11.3s→1.3s (8.6×)；n=15: 2.4s→0.4s (5.7×) |

| 5 | "聚集分布下限制影响更小" | ✅ | 1-center 全部 k=1 最优；真实地理分布 gap 更大 |
| 6 | "大 n 实例也存在少 truck 节点好解" | ✅ | FSU n=15、burma14 n=14 均在小 k 下达最优 |

## 七、时间增长趋势

```
 n=8:  DP 1ms      (基准)
n=10:  DP 8ms      (×8)
n=12:  DP 63ms     (×8)
n=14:  DP 651ms    (×10)
n=15:  DP 2,378ms  (×3.6)
n=16:  DP 11,293ms (×5)
```

DP 时间随 n 呈超线性增长，体现了 O(3ⁿ·n²) 的指数复杂度。论文表 3 中 n=18 时无限制 DP 已无法在 12 小时内完成。RDP 通过限制 k 大幅缩短时间，可求解更大 n。
